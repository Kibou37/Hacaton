# Рекомендации по выбору одежды

Веб-приложение для получения персональных рекомендаций по выбору типа и цветовых палитр одежды на основе параметров пользователя.

## Функциональность

1. **Ввод параметров пользователя:**
   - Пол
   - Рост
   - Цвет волос и глаз
   - Оттенок кожи
   - Замеры (ОГ, ОТ, ОБ)

2. **Анализ и определение:**
   - Тип фигуры (Песочные часы, Прямоугольник, Груша, Перевернутый треугольник, Яблоко)
   - Полнотная группа (XS, S, M, L, XL, XXL)
   - Цветотип внешности (Весна, Лето, Осень, Зима)

3. **Рекомендации:**
   - Типы одежды
   - Рекомендуемые силуэты
   - Длина рукавов
   - Посадка по талии
   - Цветовая палитра
   - Советы по стилю

4. **Интерактивные упражнения:**
   - Вопросы для закрепления материала
   - Проверка знаний по цветотипу и типу фигуры

## Технологии

- React 18
- TypeScript
- Vite
- CSS3

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Просмотр собранного приложения
npm run preview
```

## Структура проекта

```
src/
  components/        # React компоненты
    UserForm.tsx     # Форма ввода данных
    Recommendations.tsx  # Отображение рекомендаций
    Exercises.tsx    # Интерактивные упражнения
  utils/             # Утилиты и логика
    bodyAnalysis.ts  # Определение типа фигуры и цветотипа
    recommendations.ts  # Генерация рекомендаций
  types/             # TypeScript типы
    index.ts
  App.tsx            # Главный компонент
  main.tsx           # Точка входа
```

## Использование

1. Заполните форму с вашими параметрами
2. Получите анализ вашей фигуры и цветотипа
3. Изучите персональные рекомендации
4. Пройдите упражнения для закрепления материала
