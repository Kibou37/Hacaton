Эта папка предназначена для изображений моделей по цветотипам и полу.

Новая структура папок:

- models/
  - male/        — изображения для мужчин
    - spring/    — цветотип «Весна»
    - summer/    — цветотип «Лето»
    - autumn/    — цветотип «Осень»
    - winter/    — цветотип «Зима»
  - female/      — изображения для женщин
    - spring/
    - summer/
    - autumn/
    - winter/

Ожидаемые имена файлов внутри каждой папки цветотипа (по умолчанию .jpg):

- 1.jpg
- 2.jpg
- 3.jpg
- 4.jpg
- 5.jpg

Например:
- public/models/female/spring/1.jpg
- public/models/female/spring/2.jpg
- public/models/female/summer/1.jpg
- public/models/female/winter/1.jpg

Минимум — достаточно одного файла (например, только 1.jpg).
Приложение автоматически попытается загрузить файлы 1.jpg–5.jpg
для комбинации (пол, цветотип) и построит карусель только из реально найденных изображений.
